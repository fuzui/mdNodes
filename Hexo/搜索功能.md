<div align='center' ><font size='70'>搜索功能</font></div>

安装 `hexo-generator-searchdb`，在站点的根目录下执行以下命令：

```shell
npm install hexo-generator-searchdb --save
```

编辑 站点配置文件，新增以下内容到任意位置：

```yaml
search:
    path: search.xml
    field: post
    format: html
    limit: 10000
```
